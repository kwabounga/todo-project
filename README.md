TODO PROJECT
============

a minimalist todo app  
created with Expo / sqlite  

**installation**
```sh
git clone https://github/kwabounga/todo-project.git
npm install
# for launching expo dev server
expo start

# for building app
expo build:android
# choose apk
# and wait for build queue
```


**used packages**  
expo-sqlite  
expo-font  
@expo-google-fonts  
react-native-gesture-handler  
expo-app-loading
